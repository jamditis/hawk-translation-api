Getting started

Four steps to translate your first post

1
Install the plugin
Upload hawk-translation.zip via the WordPress plugin manager or install from the WP plugin directory.
2
Add your API key
Paste your Hawk API key in Settings → Hawk Translation. Keys follow the format:
hawk_live_xxxxxxxxxxxxxxxxxxxx
3
Configure webhook
Set the webhook URL in your Hawk dashboard so translated drafts are automatically pushed back to WordPress.
https://yournewsroom.com/wp-json/hawk/v1/receive
4
Translate a post
Open any post in the Gutenberg editor, select languages and tier in the Hawk sidebar panel, and click Translate.
