Hawk News Service — WordPress plugin

Translate from inside Gutenberg

The Hawk plugin adds a translation panel directly to the post editor, so writers can request translations without leaving the CMS.

Plugin flow

One click, ten languages

When a writer clicks Translate, the plugin posts the article to the Hawk API. Hawk translates and scores each language, then fires a webhook that creates draft posts directly in WordPress — one per language.

Interface mockup

The Hawk panel inside Gutenberg

The plugin adds a dedicated sidebar panel to the post editor. Writers choose their target languages and quality tier, then click Translate. The panel shows live job status as translations complete.

Setup

Getting started

The plugin is distributed as a standard WordPress .zip file. Once activated, enter your Hawk API key in the settings page — the translation panel appears automatically on every post.

1
Install the plugin
Upload the .zip in Plugins → Add New, or search "Hawk Translation" in the WordPress plugin directory.
2
Add your API key
Go to Settings → Hawk Translation. Paste your newsroom API key — available from your Hawk partner dashboard. hawk_live_••••••••••••••••••••ab3c
3
Set the webhook
Hawk calls your WordPress REST API when translations finish. The plugin generates a secure webhook URL automatically. /wp-json/hawk/v1/webhook
4
Publish
Translated drafts appear under the original post. Editors review and publish each language version on their own schedule.
